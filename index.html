Wolfram Alpha Usability Test

We asked users to complete 5 tasks and recorded their reactions.



Results


Task 1: Find the About Page.

Time: 8s
Difficulty: Easy

The user scrolled to the bottom of the page immediately and found it after hovering over the text for a while. They explained that the text was hard to read because it was so small.

Task 2: Ask Wolfram Alpha a Question.

Time: 10s
Difficulty: Easy

The user quickly found the text form, entered the question, and submitted it, and felt that the process was very straightforward. The majority of the time was spent typing the question out.

Task 3: Copy the Answer as Plain Text.

Time: 50s
Difficulty: Hard

The user expressed frustration and tried many methods to copy and paste the answer. They gave up before they found the “Copy as Plain Text” feature.

Task 4: Find the Examples Page.

Time: 20s
Difficulty: Moderate

The user searched the bottom of the page for the majority of the time before they found it. The user expressed frustration with small text and inconsistent content placement for page links.

Task 5: Change the Page Language.

Time: 20s
Difficulty: Moderate

The user spent most of their time searching in the top left corner as opposed to the right, but eventually clicked the language icon.

Suggestions


Font Size

The user spent most of their time searching in the top left corner as opposed to the right, but eventually clicked the language icon.

Content Placement.

Make sure the navigation links to information pages are all in the same general section of the page.
